프로젝트명
Personal Portfolio Server (PPS)

1. 만드는 목적 : 개인 프로젝트, 프로그램, 교육자료, 기술문서 등을 체계적으로 관리하고, 기업 면접 시 URL 하나만으로 포트폴리오를 확인할 수 있는 웹 기반 포트폴리오 서버를 구축한다.

2. 핵심 목표 : 
   1) 대시보드 기반의 프로젝트 현황 제공
   2) 연도별, 카테고리별 자료 관리
   3) 파일 업로드 및 다운로드 제공
   4) 프로젝트 별 상세 페이지 제공
   5) PDF, ZIP, PPT, HTML, 이미지 등 다양한 파일로 관리 가능
   6) 검색 및 태그 기능 지원
   7) 공개/비공개 설정 지원
   8) 반응형 웹 지원

3. 주요 카테고리
   1) 프로그램
   2) 교육자료
   3) 통계자료

4. 요청사항
   1) 대시보드 : 카테고리 별로 어떤 것이 몇 건이 있는지, (필요한게 있으면 너가 나에게 추천 하고 내가 컨펌하면 작업 진행) (차트도 있으면 보기 좋을 듯)
   2) 카테고리 별 관리 : 예를들어 .exe 프로그램을 등록한다면 압축하고 등록 할것이고 프로그램 사용, 설치 매뉴얼까지 등록 (PDF)할 예정이야 또한 교육자료는 PPT로 작업한 내용을 PDF로 변환하여 등록해둘 예정이고 이렇게 올린 모든 내용들을 카테고리별로 관리를 하고싶어
   2-1) 등록예시 : 프로그램 / 제작사유 / 결과 / 이미지등록화면 (클릭하면 크게 보이게)
   3) 서버 : firebase 기반 무료 서버 사용
   4) 참조할 내용 : https://github.com/trending 주소에 등록된 내용 중 우리의 프로젝트에 적합한 내용, 있으면 좋을 내용은 skills.md, claude.md, readme.md로 작성해줘
   5) 너가 작성해준 파일은 내 github에 올려두고 관리할거야 주소 : https://github.com/hiordersh/PPS
